# Documentation Directory

This folder contains the project presentation outlines, methodologies, and subproject technical writeups.

## Document Manifest

* **`00_Project_Overview.md`:** The executive summary introducing the analytical problem, core hypotheses, carriers tracked, and research findings.
* **`00_Presentation_Outline.md`:** Slide-by-slide outline designed to guide the graduation project team through their presentation to the evaluation judges.
* **`01_Excel_SubProject.md`:** Details the data prep steps, Power Pivot logical schema configuration, and dashboard design.
* **`02_SQL_SubProject.md`:** Outlines the relational schema, key constraints, table structures, and analytical SQL scripts.
* **`03_Python_SubProject.md`:** Documents the programmatic data cleaning pipeline, unit normalization parameters, and Matplotlib/Seaborn visualization setups.
* **`04_Tableau_SubProject.md`:** Details the logical relationship layer and dashboard actions used in Tableau.
* **`05_PowerBI_SubProject.md`:** Outlines the star-schema data model, calculation measures built using DAX, and report layouts.
* **`06_Recommendations_and_Conclusion.md`:** The final summary detailing actionable industry insights, policy considerations (CARES Act), and future extensions.
